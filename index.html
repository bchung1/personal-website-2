WeDJ (Solo)

WeDJ is a collaborative music application that allows multiple people to request songs to one host Spotify playlist. The host opens up their spotify playlist for music polling through a google chrome extension. Once opened, people can begin requesting songs in real time.

Built with Node.js, Socket.io, Spotify API, MongoDB, Bulma.css, JavaScript, jQuery, HTML, CSS

Github
